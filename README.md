# Metabolic Syndrome Risk Prediction

## Overview

This project focuses on predicting the risk of **Metabolic Syndrome** using supervised machine learning techniques. The objective is to compare the performance of **Naive Bayes** and **k-Nearest Neighbors (kNN)** classifiers while handling real-world clinical data containing missing values, categorical variables, and varying feature scales.

The project was completed as part of a Machine Learning assignment.

---

## Objectives

- Preprocess clinical data
- Handle missing values
- Encode categorical variables
- Scale numerical features
- Train and compare Naive Bayes and kNN models
- Perform hyperparameter tuning
- Evaluate models using multiple performance metrics
- Select the best model for Kaggle submission

---

## Machine Learning Algorithms
